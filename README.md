# Test

### Features

- Node js (Express js)
- React Routes Dom
- Bootstrap 4

### Quick Start

#### Backend

Install package node_modules
  
    npm install

Setting env configuration
  
    cp .env.example .env

Run application
  
    npm start

#### Frontend
Install package node_modules
  
    npm install

Run application
  
    npm run serve
